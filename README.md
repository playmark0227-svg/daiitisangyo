## ステータス

ドラフト（クライアント確認待ち）。確認事項への回答をもって確定版へ更新します。数値・工数は概算（仮置き）であり、法規制・税務に関わる記述は専門家確認が必要です。

## 🐟 動作するプロトタイプ（prototype/）

要件のフェーズ1コアループ（出品 → 閲覧 → 注文 → 決済(デモ) → 発送 → 管理)を実装した動作プロトタイプです。

```bash
cd prototype && npm install && npm run dev   # http://localhost:3100
```

- 3ロール（発注者・出品者・管理者）をデモアカウントで切替可能
- リアルタイム出品（写真→金額→出品の3操作）/ 定期商品テンプレート / 自動マージン計算（加算モデル）
- 在庫の売り越し防止（トランザクション）/ 締切時間管理 / アプリ内通知 / 税率区分付きCSV
- 決済はモック（本番はStripe Connectに置換予定）。DBはNode内蔵SQLite（初回起動時に自動シード）
